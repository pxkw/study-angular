+ Node.js

## QuickStart

``` sh
npm i
npm start
```
Then access http://localhost:8080 .

## Test

Run all tests:
``` sh
npm test 
```

Run unit tests:
``` sh
gulp test:unit
```


Run e2e tests:
``` sh
gulp test:e2e
```
